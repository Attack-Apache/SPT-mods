Date Made: 6/7/2024


SPT Version 3.8.3


These are the mods that I am currently using, and they should all work nicely. The names may seem odd because I use the file names instead of the site names, but don't fear they are the same. You may realize that the Realisim mod is not in this pack due to incompatibilities that would be made with some mods, and generally, I don't want to relearn Tarkov with the stuff brought with Realisim. I may make a different mod list where I have realism and use all compatible mods. Beware that this mod list will update and change over time.

I have used Merlin's SPT Mod List : https://gist.github.com/Areorah/a8dd33fa0415962883d896f825b536ba 
This inspired for most of the mods you will find in this list

If you notice that the server says that Realisim is missing JSON, go to user/mods and remove Realisim. Also, if you see the server spitting a bunch of code in yellow text about live fleas, don't worry about it :).

If you are attempting to extract the files and see that it isn't working, the rule of thumb is that if it is a .dll, it goes into BepInEx/plugins. If it is a file with stuff in it, such as Package.json, it will go into user/mods.
